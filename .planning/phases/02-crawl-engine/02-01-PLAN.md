Implement to make all tests green.

URL normalization function (module-level):
- Use urllib.parse.urlparse to decompose URL
- Lowercase scheme and netloc
- Strip trailing slash from path (unless path is just "/")
- Sort query parameters alphabetically using urllib.parse.parse_qsl and urlencode
- Remove fragment
- Remove default ports (80 for http, 443 for https)
- Reassemble with urlunparse

FrontierItem dataclass:
- url: str (normalized)
- depth: int
- ancestors: list[dict] -- each dict has keys: level (str), label (str), url (str)

URLFrontier class:
- _visited: set[str] for dedup (stores normalized URLs)
- _queues: defaultdict(deque) keyed by depth (int -> deque[FrontierItem])
- add(url, depth, ancestors) -> bool: normalize URL, check visited, add to queue if new
- pop(depth) -> FrontierItem | None: popleft from depth queue or None
- has_pending(depth) -> bool: check if depth queue is non-empty
- visited_count property: len(_visited)
  </action>

src/scraper/fetcher.py:
- Fetcher Protocol: async fetch(url, wait_selector=None, timeout=15.0) -> str | None, async close() -> None
- PlaywrightFetcher:
  - __init__(self): stores browser/context as None (lazy init)
  - async start(self): launch chromium headless, create context
  - async fetch(self, url, wait_selector=None, timeout=15.0) -> str | None:
    - Create page from context
    - goto with wait_until="domcontentloaded", timeout=30000
    - If wait_selector: wait_for_selector with timeout*1000
    - Else: wait_for_load_state("networkidle", timeout=timeout*1000)
    - On TimeoutError: log warning, get whatever content is available
    - Get page.content(), close page, return HTML
    - On any other exception: log error, close page, return None
  - async close(self): close context and browser
- HttpxFetcher:
  - __init__(self): create AsyncClient with Timeout(30.0, connect=10.0, read=30.0)
  - async fetch(self, url, **kwargs) -> str | None:
    - response = await client.get(url)
    - If 429: raise RateLimitError(response)
    - If 5xx: raise ServerError(response)
    - If 404: log and return None
    - Return response.text
  - async close(self): close client
- Define RateLimitError and ServerError exception classes (used by retry logic in Plan 02)

Use structlog for logging (bound logger per class). Import from Phase 1's logging setup if available, otherwise use structlog.get_logger() directly.
