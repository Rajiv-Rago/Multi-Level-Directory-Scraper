    - clean_text_fields decodes HTML entities: "Caf&eacute; &amp; Bar" -> "Cafe & Bar"
    - clean_text_fields collapses internal whitespace: "Foo   Bar\n\tBaz" -> "Foo Bar Baz"
    - clean_text_fields preserves None values (does not crash on None)
    - clean_text_fields applies to all text fields: name, address, description
    - clean_text_fields returns same number of records as input

    - normalize_phones converts "+44 20 8366 1177" to "+442083661177" (international already has country code)
    - normalize_phones keeps invalid number "not-a-phone" as-is and adds warning to collector
    - normalize_phones keeps empty/None phone values unchanged
    - normalize_phones records stat "phones_normalized" with count of successful normalizations
    - normalize_phones records stat "phones_failed" with count of failures
    - normalize_phones adds warning with field="phone", reason containing "parse" or "invalid"
  </behavior>

       - Use `record.model_copy(update={...})` to create new records with cleaned text fields (name, address, description)
       - Do NOT clean phone, website, source_url (those have their own stages)
       - Add stat "records_cleaned" with count

       - On NumberParseException or invalid number: return original string, False
       - Collector gets add_warning for each failure (field="phone", value=original, reason=specific error, source_url=record.source_url)
       - Collector gets add_stat("phones_normalized", count) and add_stat("phones_failed", count)

<task type="auto" tdd="true">
  <name>Task 3: URL validation and deduplication stages</name>
  <files>src/pipeline/urls.py, src/pipeline/dedup.py, tests/test_pipeline/test_urls.py, tests/test_pipeline/test_dedup.py</files>
  <behavior>
    URL validation (test_urls.py):
    - validate_urls resolves relative URL "/about" with base "https://example.com/page" to "https://example.com/about"
    - validate_urls resolves "../other" with base "https://example.com/dir/page" to "https://example.com/other"
    - validate_urls keeps absolute URL "https://example.com/page" unchanged
    - validate_urls flags "not-a-url" as invalid (adds warning) but preserves it in output
    - validate_urls preserves None website values
    - validate_urls records stat "urls_resolved" and "urls_invalid"
    - validate_urls adds warning with field="website" for invalid URLs

    Deduplication (test_dedup.py):
    - deduplicate removes duplicate where two records have same normalized name+address
    - deduplicate keeps the record with more complete fields (fewer None values)
    - deduplicate uses source_url as fallback key when address is None
    - deduplicate normalizes keys: lowercase, strip whitespace, remove punctuation ("Café Bar" and "cafe bar" match)
    - deduplicate logs each dedup decision with both source_urls in collector warnings
    - deduplicate records stat "duplicates_removed" with count
    - deduplicate returns fewer records than input when duplicates exist
    - deduplicate preserves all records when no duplicates exist
  </behavior>
  <action>
    1. Create `src/pipeline/urls.py` with `validate_urls(records: list[DirectoryRecord], collector: ValidationCollector) -> list[DirectoryRecord]`:
       - Internal helper `resolve_and_validate_url(url: str | None, base_url: str) -> tuple[str | None, bool]`
       - Uses urllib.parse.urljoin(base_url, url) for resolution (base_url comes from record.source_url)
       - Uses urllib.parse.urlparse() to validate: scheme in ('http', 'https') and netloc is non-empty
       - Invalid URLs: keep original value, add warning (field="website", reason="Invalid URL format")
       - Stats: "urls_resolved" for successful resolutions, "urls_invalid" for invalid

    2. Create `src/pipeline/dedup.py` with `deduplicate(records: list[DirectoryRecord], collector: ValidationCollector) -> list[DirectoryRecord]`:
       - Internal helper `make_dedup_key(record: DirectoryRecord) -> str`: if address exists, key = normalize_for_dedup(name) + "|" + normalize_for_dedup(address); else key = record.source_url
       - Internal helper `normalize_for_dedup(text: str) -> str`: unicodedata.normalize("NFC", text).casefold(), re.sub(r'[^\w\s]', '', ...), re.sub(r'\s+', ' ', ...).strip()
       - Internal helper `completeness_score(record: DirectoryRecord) -> int`: count non-None fields
       - Group by dedup key. For each group with >1 record: keep record with highest completeness_score, add warning for each removed record (field="duplicate", value=removed record name, reason="Duplicate of {kept record name}", source_url=removed record source_url). Also log kept record's source_url in reason.
       - Stats: "duplicates_removed" count

    3. Write RED tests first, then GREEN. Use parametrize for URL resolution edge cases.
  </action>
